# Contributing

Contributions should keep this repository useful, factual, and safe to share.

## What belongs here

- Original summaries of NetSuite and SuiteScript behavior
- Practical recommendations based on documented behavior
- Small examples written specifically for this repository
- Links to official Oracle NetSuite documentation

## What does not belong here

- Copied Oracle/NetSuite documentation pages
- Customer-specific scripts, account IDs, internal URLs, screenshots, or data
- Credential material, tokens, secrets, or private implementation details
- Claims that are not sourced or clearly marked as experience-based notes

## Style

- Link to official sources where possible.
- Keep quoted text short and necessary.
- Prefer dates and NetSuite release numbers for time-sensitive behavior.
- Use clear caveats when behavior depends on account preferences, script type, or
  runtime version.
